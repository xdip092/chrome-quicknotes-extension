# Chrome Quick Notes Extension

A minimal Chrome extension (Manifest V3) to store quick notes using `chrome.storage.local`.

## Load in Chrome

1. Open `chrome://extensions/`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select this folder: `projects/chrome-quicknotes-extension`

## Files

- `manifest.json`
- `popup.html`
- `popup.css`
- `popup.js`
- `service-worker.js`
